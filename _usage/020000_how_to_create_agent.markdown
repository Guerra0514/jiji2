「エージェント」とは、<b>レート情報を受け取って取引やPush通知などの処理を行うRubyのクラス</b>です。 <br/>
プログラミングにより、以下の処理を行なうことが可能です。

- 移動平均など各種シグナルの計算
- シグナルに基づく取引の実行(売り・買い・決済)
- グラフの描画
- Push通知、メールでのメッセージ送信
- ユーザーから指定されたアクションの実行
- ログの出力(デバッグ用)

Jiji では　`Jiji::Model::Agents::Agent` モジュールをインクルードしたクラスがエージェントとして認識されます。
