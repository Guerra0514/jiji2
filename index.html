誰でも、いますぐ、かんたんにFX自動取引を開始できます

いますぐ、インストール

※「必ず利益が出せる取引アルゴリズム」を提供するソフトウェアではありません。
取引アルゴリズムを作成・検証し、運用を行うためのフレームワークです。
※スマホアプリは有料のサービスです。

3ステップで、システムトレードを開始。

1. 取引アルゴリズムを作成

  • 自分だけの取引アルゴリズムを、オブジェクト指向プログラミング言語「Ruby」で作成。
  • 添付ライブラリを使って、移動平均やRSIを使ったアルゴリズムをかんたんに作成できます。
  • メールやPush通知の送信にも対応。

2. バックテストで検証

  • 過去10年のデータを使って、取引アルゴリズムの検証が可能。
  • 勝率やProfitFactorなどアルゴリズムの成績は、UIで詳細に分析することができます。

3. 24時間、自動取引

  • OANDA Japan の口座で、作成した取引アルゴリズムを実行。 (発注や決済には OANDA FX REST API を利用しています)
  • 取引の状況は、スマホアプリまたはWebUI でいつでもどこでも確認可能。
  • OANDA Japan の無料デモ口座にも対応。まずは気軽にお試しできます。

裁量トレードにも使える、"Bot型"取引アルゴリズム

Jijiでは、取引アルゴリズムからメールやPush通知を送信することができます。
また、アルゴリズムに命令を送って既定のアクションを実行させることも可能。
これらを組み合わせて、裁量トレードをアシストするBot型の取引システムを構築することもできます。

≪使い方イメージ≫

  • 取引アルゴリズムが、レートの動きを監視。
  • レートがボリンジャーバンド -2αを下回ったところで、Push通知を送信。
  • Push通知を受け取り、次のアクションを人間が判断。
  • アルゴリズムが命令を受け取り、アクションを実行します。

クラウド対応。なので、運用も低コスト。

クラウドプラットフォームのHerokuに対応。1クリックでインストールして利用を開始することができます。
Docker Imageを配布しているので、ローカルマシンへのインストールも簡単。AWSなどDockerに対応した他のクラウドサービスも利用できます。

  • Herokuなら、月額$7で24時間連続稼働するシステムが構築可能。
  • システムの管理は、WebUIまたはスマホアプリから行えます。
  • もちろん、自宅のPCにインストールして使うことも可能。
  • AWSなどDockerに対応した他のクラウドサービスも利用できます。
